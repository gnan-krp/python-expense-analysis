# Expense Tracker Data Analysis (Python)

This project performs basic data analysis on user-entered expense data using core Python only.

## Features
- Accepts expense data from user input
- Calculates total and average expenses
- Finds highest and lowest expenses
- Identifies days with above-average spending
- Categorizes spending behavior

## Technologies Used
- Python (Core)

## Learning Outcome
This project strengthens logical thinking, data analysis fundamentals, and prepares for advanced tools like Pandas.
